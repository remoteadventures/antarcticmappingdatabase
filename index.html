Welcome to the Antarctic Mapping Database

Welcome to the Antarctic Mapping Database mapping portal. This takes the SCAR Antarctic Placename database and maps the locations — and if you are in Antarctica, you can see what's nearby!

Buttons across the top

☰ List toggleShows or hides the place name list panel. On small screens, use this to switch between map and list views.
Search barFilter by name in real time. Use the SCAR / COMNAP toggle to switch between the 40,000+ SCAR place name database and the 114 COMNAP research facilities.
Mercator / PolarSwitch between standard scrolling map and Antarctic Polar Stereographic view (EPSG:3031). Polar view correctly shows the full continent including the South Pole. Mercator cuts off at 85°S.
⧉ LayersOpens the layer panel. Switch base maps (Esri Satellite, Google Satellite), toggle overlays (SCAR dots, COMNAP facilities, coastlines, territorial claims, contours, lat/lon grid, sea ice, wind), and import your own files. Imported files appear here with visibility toggles and opacity sliders.
⊕ CrosshairToggle a crosshair at the map centre. The latitude and longitude update live at the bottom of the screen as you pan.
◎ My locationCentre the map on your current GPS position, marked with an orange dot.
⊙ Closest to meUses your GPS to find and sort nearby places by distance. A radius slider appears — default 50 km, up to 3,000 km. Works with both SCAR and COMNAP databases.
ContoursTwo contour layers: Low res (fast, always available offline) and Medium res (37MB, downloads on first use, requires internet). Both show ADD Antarctic contour data with heights on hover.
File importImport GeoJSON, KML, KMZ, GPX or GeoTIFF files via the + Add file button in the Layers panel. Imported files appear with visibility toggles and opacity sliders. GeoTIFFs display in Mercator view only; LERC-compressed files need re-exporting from QGIS first (Raster → Convert → Cloud Optimised GeoTIFF, Deflate compression).
Place markersBlue dots = SCAR place names. Orange dots = COMNAP facilities. Tap any dot for full details. Nearby SCAR dots cluster at low zoom — tap a cluster to zoom in.
